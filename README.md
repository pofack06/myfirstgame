# myfirstgame
<h1><b>Приключение Лунного кота (Pygame)</b></h1>
<h4><b>Платформер в стиле Mario с боями против врагов и боссов.</b></h4>
<h4>Игрок управляет персонажем, сражается с мобами, собирает бонусы и побеждает боссов на нескольких уровнях.</h4>

<h2><b>🎮 Особенности</b></h2>
<ul>
<h4>✔️ Классический 2D-платформер с элементами экшена.</h4>
<h4>✔️ Несколько уровней с уникальными механиками.</h4>
<h4>✔️ Боссы с разными типами атак.</h4>
<h4>✔️ Система здоровья, атаки и бонусов.</h4>
<h4>✔️ Плавная анимация и параллакс-фон.</h4>
</ul>
<h2><b>⚙️ Установка и запуск</b></h2>
<h4><b>Требования</b></h4>

<li>Python 3.8+
<li>Pygame (pip install pygame)</li>
</ul>

<h4>Запуск</h4>
<h5><li>Клонируйте репозиторий:</li></h5>


  ```bash
  git clone https://github.com/pofack06/myfirstgame.git
  cd platformer-game
  ```
<h5><li>Запустите игру:</li></h5>

  ```bash
  main.py
  ```

<h2><b>🕹 Управление</b></h2>
<ul><h4>
  <h4><h>Клавиша Действие</h></h4>
<h4>← →	Движение влево/вправо</h4>
<h4>↑ / Пробел	Прыжок</h4>
<h4>Z	Атака</h4>
<h4>X	Спец. способность</h4>
<h4>ESC	Пауза / Меню</h4>
</ul>

<h2>🎨 Графика и звуки</h2>
<ul><h4>Спрайты: Pixel-арт (самодельные или с OpenGameArt).</h4>

<h4>Звуки: Эффекты ударов, прыжков, фоновая музыка.</h4></ul>
